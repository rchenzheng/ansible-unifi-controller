Ansible Role: Unifi Controller
=========

Installs unifi controller

Requirements
------------

Currently only supported by Debian stretch and most Ubuntu distributions due to openjdk8

Dependencies
------------

- openjdk 8

Example Playbook
----------------

    - hosts: servers
      roles:
         - rchenzheng.unifi_controller
